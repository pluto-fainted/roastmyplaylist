# RoastMyPlaylist

Enter your songs. Get destroyed by a roatster.

## Whats this?

You type in your tops songs, pick a character personality you wanna get destroyed by, and it brutally roasts your music tast. Sharable card included so you can spread the pain.

## Roaster Mode

- Gordon Ramsay — treats your playlist like a dish that failed
- Gen Z Bestie — absolutely horrified, no cap
- Music Snob — condescending, references artists you've never heard of  
- Unhinged AI — chaotic, somehow still devastating

## Built With

- HTML / CSS / Vanilla JS
- Claude API (for the actual roasting)
- html2canvas (sharable image cards)
- Vercel (deployment)

## How to locally run this?

just open index.html in your browser. or clone it:
